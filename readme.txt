== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `software-requirements.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php echo do_shortcode('[WPreq_soft]');?>` in your theme
